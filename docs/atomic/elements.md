# Elements (Atoms)

Éléments HTML de base autonomes qui ne composent pas d'autres composants.

**Total** : 24 composants  
**Implémentés** : 21/24

---

## Text & Content

**Heading** - H1-H6 avec hiérarchie et variantes  
Usage : Titres principaux, titres de sections (hero, services)  
Implémenté : ✅

**Text** - Paragraphe et texte inline avec variantes  
Usage : Descriptions, métadonnées de biens  
Implémenté : ✅

**Link** - Lien hypertexte avec icône optionnelle  
Usage : "Voir tous", "Voir la sélection", "En savoir plus"  
Implémenté : ✅

**Eyebrow** - Petit label au-dessus d'un titre  
Usage : Info bien (Prix, Surface, Référence)  
Implémenté : ✅

## Media

**Icon** - Système d'icônes SVG avec sprite  
Usage : Icônes UI, décoratives, interactives  
Implémenté : ✅

**Image** - Image responsive avec ratios  
Usage : Photos de biens, images hero  
Implémenté : ✅

**Avatar** - Image profil avec fallback initiales  
Usage : Profils agents/conseillers  
Implémenté : ✅
