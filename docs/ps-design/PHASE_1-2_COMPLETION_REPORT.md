## 🎓 Learnings

### For Future Icon System Changes
1. Use `npm run build` (not `npm run build:icons` - that script doesn't exist)
2. Biome formatting can be tricky with generated JSON - use `.biomeignore`
3. Registry categorization works best with naming conventions
4. Always verify file paths match between constants in build-icons.mjs

### For Component Developers
1. All 141 icons now available via `data-icon="name"` in HTML
2. Icon names visible in generated registry.json
3. No more "icon not available" errors
4. Can freely add new icons - just drop SVG in icons-source/, run build
