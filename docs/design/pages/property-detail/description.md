# Description

Bloc de texte avec contenu long et bouton « Voir plus ».

## Modèle de contenu
- Intro: `string`
- Corps: `string`

## UX
- Tronquer le texte avec « Voir plus » pour développer.

## Accessibilité
- Le bouton indique l’état étendu (`aria-expanded`).

## Tokens
- Échelle typographique; espacement entre paragraphes.
