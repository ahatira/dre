# Biens similaires

Carrousel/grille d’offres liées.

## Modèle de contenu
- Éléments : `array<Card>` (image, titre, prix, localisation, CTA)

## UX
- 4 colonnes desktop, 2 colonnes tablette, 1 colonne mobile ; contrôles précédent/suivant.

## Accessibilité
- Contrôles du carrousel avec libellés ; cartes avec texte alternatif.

## Tokens
- Espaces de grille ; espacement des cartes ; couleurs sémantiques pour les CTA.
