# Calculateur de temps de trajet

Saisie d’une adresse d’origine ; calcul du temps de trajet sur la carte.

## Modèle de contenu
- Origine : `string`
- Mode : `enum` (car, transit, walk)
- Résultat : `string` (ETA)

## UX
- Champ simple + bouton de calcul ; afficher le résultat sous la carte.

## Accessibilité
- Champ avec libellé ; résultat annoncé.

## Tokens
- Tailles d’input ; espacement autour du résultat.
