# Section Énergie

Deux widgets : consommations énergétiques, émissions de gaz à effet de serre.

## Modèle de contenu
- Consommations: valeur d’échelle + libellé; indicateur fourni
- Émissions: valeur d’échelle + libellé; indicateur fourni

## UX
- Utiliser une échelle visuelle cohérente; afficher l’état « Non fourni ».

## Accessibilité
- Les graphiques doivent avoir des équivalents textuels.

## Tokens
- Couleurs sémantiques d’information; tokens de bordure/overlay.
