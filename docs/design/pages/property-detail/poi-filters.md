# Filtres POI

Groupe de cases à cocher pour activer/désactiver des catégories sur la carte.

## Modèle de contenu
- Catégories : `array<{key:string, label:string}>`

## UX
- Cases à cocher en ligne ; mise à jour instantanée de la carte.

## Accessibilité
- Groupe avec libellé ; chaque case à cocher possède un libellé.

## Tokens
- Espacement des cases ; couleurs neutres.
