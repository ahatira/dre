# Carte consultant

Résumé du conseiller avec avatar, nom, téléphone, CTA de contact.

## Modèle de contenu
- Nom: `string`
- Rôle: `string`
- Téléphone: `string`
- Email: `string`
- Photo: `image`
- CTA contact: ouvre le modal de formulaire de contact

## UX
- Carte compacte; CTA placé près du téléphone.

## Accessibilité
- Lien téléphone en `tel:`; CTA avec label clair.

## Tokens
- Tailles d’avatar, typographie, espacement.
