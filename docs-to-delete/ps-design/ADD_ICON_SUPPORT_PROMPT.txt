Je veux ajouter le support des icônes (data-icon) directement dans les composants comme une fonctionnalité native.

Utilise le prompt: docs/ps-design/ICON_STORY_FIX_PROMPT.md

**Composants à modifier**: Button, Badge, Alert (et autres ensuite)

**Modifications requises pour chaque composant**:

1. **{component}.yml**: Ajoute les paramètres
   icon:
     description: "Icon name (e.g., 'check', 'arrow-right'). Leave empty for no icon."
     type: string
     default: null
   
   iconPosition:
     description: "Icon position: 'start' (::before, default) or 'end' (::after)"
     type: string
     default: start
     options:
       - start
       - end

2. **{component}.twig**: Ajoute les attributs dans l'élément principal
   {% if icon %}data-icon="{{ icon }}"{% endif %}
   {% if iconPosition != 'start' %}data-icon-position="{{ iconPosition }}"{% endif %}

3. **{component}.stories.jsx**: Ajoute les argTypes
   icon: {
     description: 'Icon name from sprite (no "icon-" prefix, e.g., "check", "arrow-right")',
     control: { type: 'select' },
     options: [null, ...iconsRegistry.names],
     table: { category: 'Content', type: { summary: 'string' } },
   },
   iconPosition: {
     description: 'Icon position: start (::before, default) or end (::after)',
     control: { type: 'inline-radio' },
     options: ['start', 'end'],
     table: { category: 'Appearance', type: { summary: 'start | end' }, defaultValue: { summary: 'start' } },
   },
   
   Et ajoute une story exemple:
   export const WithIcon = {
     args: { label: 'Proceed', icon: 'arrow-right', iconPosition: 'end' },
   };

4. **CSS**: Aucune modification - icons.css gère déjà tout automatiquement

**Validation**:
✅ npm run build (doit passer)
✅ npm run watch (vérifier visuellement)
✅ Commit: feat({scope}): Add icon support to {component}

**Ressources**:
- Icons disponibles: source/patterns/documentation/icons-registry.json (141 icônes)
- Référence CSS: source/props/icons.css (gère display: inline-flex + gap + position)
- Exemple complet: source/patterns/elements/button/button.twig (voir structure attributes)

Commence par Button, puis Badge, puis Alert. Fais un composant à la fois, teste et commit avant le suivant.
