# Documentation Archive

This folder contains archived documentation from completed phases and migrations.

## Structure

- **`audits/`** - Compliance audits and component validations
- **`icon-migration/`** - Complete icon system overhaul documentation (Dec 2025)
- **Root archive files** - Completed standardization and update reports

## Why Archive?

These documents served their purpose during active development phases but are no longer needed for daily reference. They're preserved for:

- Historical context
- Audit trails
- Lessons learned for future work
