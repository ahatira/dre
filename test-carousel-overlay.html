Carousel - Pixel Perfect Overlay Style

Specifications Pixel Perfect

  • Boutons navigation: Circulaires 48px (--size-12), blanc (#fff), ombre (--shadow-3), chevrons verts (--ps-color-primary-600)
  • Position boutons: Absolute overlay, centré verticalement (top: 50% + translateY(-50%)), padding horizontal 16px (--size-4)
  • Pagination: Absolute overlay, bottom 24px (--size-6), bullets 12px (--size-3), blancs avec ombre (--shadow-2)
  • Bullet actif: Vert (--ps-color-primary-600), ombre renforcée (--shadow-3)
  • Image: Pleine largeur, sans border-radius, aspect ratio préservé
  • Hover: Boutons scale 1.05 + ombre renforcée (--shadow-4), bullets scale 1.15
